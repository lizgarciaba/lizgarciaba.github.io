Comprime tus imágenes para Web

En el mundo web y móvil es muy importante la velocidad con la cual cargan los sitios, y eso viene en parte por lo optimizado que se encuentren respecto al peso que tengan nuestras imágenes del sitio. Una buena compresión de imágenes nos puede dar buenos frutos de tiempo de carga de nuestro portal.

Les muestro una aplicación bastante sencilla para comprimir imágenes llamada CompressNow. Se trata de un servicio web cuya función es comprimir el peso de las imágenes, podemos llegar a comprimir imágenes de 6mb hasta llevarlas a 600kb.

Como usar CompressNow

El servicio funciona de manera muy sencilla, simplemente entramos a la web, seleccionamos la imagen que deseamos reducir el tamaño, luego seleccionamos el porcentaje que deseamos que se reduzca y listo. Clic en Compress Now y la imagen se reduce.

Ahora simplemente, hacemos clic derecho a la imagen de la derecha y la descargamos, así tendremos nuestra imagen reducida de manera sencilla.
